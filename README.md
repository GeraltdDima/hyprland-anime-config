### 1.Dependencies

You must have linux with ge `hyprland`.

Packages -
1. `hyprpaper`
2. `hyprlock`
3. `wlogout`
4. `waybar`
5. `noto-fonts`
6. `ttf-jetbrains-mono`
7. `ttf-font-awesome`

### 2.Installation 

`git clone https://github.com/GeraltdDima/hyprland-anime-config.git ~/`
`cd ~/hyprland-anime-config`
`cp hypr ~/.config -r`
`cp waybar ~/.config -r`
`cp wofi ~/.config -r`
`cp wlogout ~/.config -r`
`cp BashScripts ~/ -r`
`cp Icons ~/ -r`
