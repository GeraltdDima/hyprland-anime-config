### 1.Dependencies

You must have linux with ge `hyprland`.

Packages -
1. `hyprpaper`
2. `hyprlock`
3. `wlogout`
4. `waybar`
5. `noto-fonts`
6. `ttf-jetbrains-mono`
7. `ttf-font-awesome`

### 2.Installation -

1. `git clone https://github.com/GeraltdDima/hyprland-anime-config.git ~/`
2. `cd ~/hyprland-anime-config`
3. `cp hypr ~/.config -r`
4. `cp waybar ~/.config -r`
5. `cp wofi ~/.config -r`
6. `cp wlogout ~/.config -r`
7. `cp BashScripts ~/ -r`
8. `cp Icons ~/ -r`
